# НИРС — Тематическое моделирование

## Тема
Сравнительный анализ методов тематического моделирования
на корпусе русскоязычных текстов: TF-IDF, LDA, NMF
с оценкой качества через когерентность.

## Датасет
Lenta.ru, ~10 000 статей. Поля: title, text, topic, date.

## Стек
Python 3.12, PostgreSQL, pandas, numpy, scikit-learn,
gensim, pymorphy2, matplotlib, seaborn, plotly, python-dotenv

## Архитектура
data/        — сырые данные (не коммитить)
notebooks/   — Jupyter ноутбуки
sql/         — DDL и SQL-запросы
src/         — Python-модули
tests/       — тест-кейсы
reports/     — графики и таблицы
docs/        — логи сессий

## База данных
Три таблицы:
- topics (id, name)
- raw_documents (id, topic_id, title, body, published_at, word_count, char_count)
- processed_documents (id, doc_id, lemmas, tokens_raw, lang)

## Алгоритмы (по очереди)
1. TF-IDF — ключевые слова по рубрикам
2. LDA    — тематическое моделирование, когерентность
3. NMF    — сравнение с LDA, когерентность

## Метрики сравнения
| Метод | Когерентность | Время обучения | Интерпретируемость | Вывод |
